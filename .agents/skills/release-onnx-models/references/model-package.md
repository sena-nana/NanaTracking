# Model package

Required paths:

- `model.onnx`
- `schema.json`
- `signal-registry-revision.json`
- `normalization.json`
- `runtime-metadata.json`
- `calibration-schema.json`
- `adapter-contract.json`
- `test-vectors/`

Metadata records checkpoint and model digests, opset, input shape/layout/color assumptions, named
outputs, precision support, temporal state, NTP and feature revisions, and smoke/production status.
